# Quiz-App
An interactive and user-friendly Quiz Application built using HTML, CSS, and JavaScript. This app presents multiple-choice questions one at a time, allowing users to select answers and track their performance throughout the quiz.

Each question includes a countdown timer to increase engagement and challenge, while a progress indicator keeps users informed of their position in the quiz. At the end, the app displays the final score along with an option to restart and try again.

The interface is designed with a clean and modern look, enhanced by smooth animations and a fully responsive layout that works seamlessly across mobile, tablet, and desktop devices.

Key Features:

Multiple-choice questions displayed one at a time
Timer for each question
Progress indicator
Final score display
Restart quiz option
Smooth animations and transitions
Fully responsive design
